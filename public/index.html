Recipes API
Postman collection url

Simple REST service providing working API for demo applications written by interviewed front-end developers.

A1 GET /api/recipes/

Recipes listing, can be used with ?s= query parameter to select resulting object's fields, ex: /api/recipes/?s=title+minutes

A4 PUT /api/recipes/{RECIPE_ID}

Updates recipe by id

A5 DELETE /api/recipes/{RECIPE_ID}

Removes recipe by id

B1 POST /api/recipes/{RECIPE_ID}/photos

Uploads new photo for recipe. Expects content type 'multipart/form-data' and 2 body parts:
  • recipePhoto containing image bytes in jpeg or png format
  • (optional) recipePhotoTitle with image title text

New PHOTO_ID can be parsed from returned Link header.

B2 GET /api/recipes/{RECIPE_ID}/photos/{PHOTO_ID}

Gets recipe's photo

B3 DELETE /api/recipes/{RECIPE_ID}/photos/{PHOTO_ID}

Removes photo from recipe
