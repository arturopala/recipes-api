Siili Recipes API

GET /api/recipes/

Recipes listing, can be used with ?s= query parameter to select resulting object's fields, ex: /api/recipes/?s=title+minutes

POST /api/recipes/

Create new recipe, title must be unique

GET /api/recipies/{ID}

Get recipe by its unique id

PUT /api/recipies/{ID}

Update recipe by id

DELETE /api/recipies/{ID}

Remove recipe by id
